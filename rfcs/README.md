# RFCs

This directory holds protocol design notes that are more focused than the broad
draft in `agent-workspace.md` and more stable than scratch implementation
journal entries in the workspace repo.

Conventions:
- file names use `NNNN-short-title.md`
- statuses are `draft`, `proposed`, `accepted`, `superseded`, or `rejected`
- each RFC should describe the problem, the decision, and the main tradeoffs
